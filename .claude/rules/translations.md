# Translation Rules

- Ogni tool DEVE avere traduzioni in TUTTE le 6 lingue: IT, EN, ES, FR, DE, PT
- Le traduzioni vanno nell'oggetto `tools` E in `getToolsByCategory()`
- SEO article: 300-400 parole per lingua, keyword naturali
- Meta tags: title e description in tutte le lingue
- FAQ: 4-5 domande per lingua
- NO stringhe hardcoded nel codice — tutto in translations.ts
